# 🎂 Memory Constellation

An interactive cinematic birthday experience built to celebrate friendship, unforgettable moments, and years of memories.

## ✨ Overview

Memory Constellation is more than a birthday website.

It is a digital journey through memories, adventures, laughter, and friendship. The project combines cinematic storytelling, immersive animations, interactive experiences, and emotional design to create a unique birthday surprise.

Visitors can explore memories through multiple storytelling sections including timelines, memory galaxies, scrapbooks, letters, and interactive experiences.

## ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/rajeshgupta0/bithday.git
```

Navigate to the project:

```bash
cd bithday
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
